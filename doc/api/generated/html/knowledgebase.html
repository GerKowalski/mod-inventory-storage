get /knowledge-base/instance

Retrieve a list of instance items.

Headers

  • Authorization: required (string)

    Used to send a valid JWT token.

Query Parameters

  • lang: (string - default: en - pattern: [a-zA-Z]{2})

    Requested language. Optional. [lang=en]

  • Partial Title: (string)

    text to search for within the title of instances

    Example:

    partialTitle="Advanced Manufacturing"
  • Identifier: (string)

    identifier to search instances for

    Example:

    isbn="9781466636897"

post /knowledge-base/instance

Create a new instance item.

  • Request
  • Response

Headers

  • Authorization: required (string)

    Used to send a valid JWT token.

    Example:

    Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiIxMjM0NTY3ODkwIiwibmFtZSI6IkpvaG4gRG9lIiwiYWRtaW4iOnRydWV9.TJVA95OrM7E2cBab30RMHrHDcEfxjoYZgeFONFh7HgQ

Query Parameters

  • lang: (string - default: en - pattern: [a-zA-Z]{2})

    Requested language. Optional. [lang=en]

Body

Type: application/json

Schema:

{
  "$schema": "http://json-schema.org/draft-04/schema#",
  "type": "object",
  "properties": {
    "title": {
      "type": "string"
    },
    "identifiers": {
      "type": "array",
      "minItems": 1,
      "items": {
        "type": "object",
        "properties": {
          "namespace": {
            "type": "string"
          },
          "value": {
            "type": "string"
          }
        },
        "required": [
          "namespace",
          "value"
        ]
      }
    }
  },
  "required": [
    "title"
  ]
}

Example:

{
  "title": "ADVANCING LIBRARY EDUCATION: TECHNOLOGICAL INNOVATION AND INSTRUCTIONAL DESIGN",
  "identifiers": [{
    "namespace": "isbn",
    "value": "9781466636897"
  }, {
    "namespace": "ybp",
    "value": "1"
  }]
}
